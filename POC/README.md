# Roll

**从"那个谁"到"有人"——大学四年的互动叙事游戏。**

## 一句话

一个关于"记不住名字的大学同学"的互动叙事游戏。军训里你记不住每个人的脸和名字——直到你和某个人产生了连接，ta 才从"那个谁"变成一个具体的人。

## 核心玩法

### 1. 场景探索式交互
不是"三个按钮选一个"。场景里每个元素都能点——太阳、教官、旁边的同学、地上的蚂蚁。点哪里，触发什么故事。**选择从探索中自然浮现。**

### 2. 跨场景标签系统
场景 1 的行为生成标签（内向/勇敢/爱发呆），影响场景 2 和 3 的叙事走向。**你选的不是选项，是这个人会成为谁。**

### 3. 节奏游戏
拉歌对唱和阅兵训练用节奏游戏呈现。不是装饰，是隐喻——融入集体的节奏感。

### 4. 揭面结算
玩完后：**章末过场**里看清本章认识的人；**遗憾清单**里看见你可能错过的——前置了解大学，减少真实遗憾。

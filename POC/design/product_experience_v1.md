# 产品体验总纲 v1

> **定调日期**：2026-07-09  
> **适用**：Roll /「那个谁」全产品（军训篇为首个大场景）  
> **关联文档**：`产品定义.md`、`visual_guide_v2.md`、`scene_new_chapters_v1.md`

---

## 一、产品要解决的问题

**不是**帮玩家「选对」，**而是**在玩的过程中，对**大学生活更真、更全**地建立体感，从而减少真实入学后的盲目与遗憾。

| 维度 | 说明 |
|------|------|
| **谁** | 准大学生、大一新生（核心）；高年级可选对应阶段章节 |
| **选什么场景** | 只有**大一/大学特有**的情境——换身份不会有同样的感受（见下方自检） |
| **玩完带走什么** | 对「大学原来是这样」的具体认知 + 对自己选择模式的觉察 + **遗憾清单**带来的前置反思 |
| **语气** | 不说教、不鸡汤；用细节、后果、平行路径呈现「你可能错过的」 |

### 场景入选自检（每条必过）

1. 高中生会经历吗？→ 会则不做  
2. 打工人会经历吗？→ 会则不做  
3. 玩家能否在 8～15 分钟内感到「靠，这不就是……」？→ 不能则改  
4. 选错/选怂是否有**可感知的代价**（不必惩罚感，但要真实）？

## 二、结构：大场景 → 子场景 → 章末过场 → 全剧高潮

```
【全游戏】
  │
  ├─ 大场景 1：军训篇（9月）
  │     ├─ 子场景 S1 站军姿
  │     ├─ 子场景 S2 拉歌
  │     ├─ 子场景 S3 领队选拔（含节奏关）
  │     └─ ★ 章末过场：小动画 + 本章羁绊揭面
  │
  ├─ 大场景 2：百团大战（10月）
  │     └─ … → ★ 章末过场
  │
  ├─ 大场景 3：离家过节 …
  │
  └─ ★ 全剧终：遗憾清单（情感高潮）→ 可选主角揭面 → 重玩/分享

【军训试玩版】仅含大场景 1；全剧终展示「本章遗憾清单」+ 全作预告
```

| 层级 | 名称 | 时长（参考） | 玩家得到什么 |
|------|------|-------------|-------------|
| 子场景 | 站军姿 / 拉歌 / … | 2～4 分钟 | 探索、选择、即时后果 |
| **大场景** | 军训篇 / 百团大战 / … | 8～15 分钟 | 一段完整大学记忆弧线 |
| **章末过场** | Interlude | 15～30 秒 | 情绪收束 + **看清本章认识的人** |
| **全剧终** | 遗憾清单 | 1～2 分钟 | **全路径反思高潮**（试玩版为本章清单） |

## 三、章末过场（Interlude）

### 3.1 定位

- 每个**大场景结束后**播放，**不是**每个子场景都播  
- 承担：**时间跳跃** + **本章记忆落袋** + **羁绊揭面**  
- **不承担**：完整遗憾清单（留给全剧终）

### 3.2 内容结构（固定三段）

```
[1] 过场动画（5～15s）
    实景手绘风；无对白或仅一句旁白
    例·军训：「解散哨响了。操场空了。你后来再也没站过那么久的军姿。」

[2] 本章羁绊揭面（按角色依次，各 2～3s）
    仅揭示【本大场景内】建立羁绊的 NPC
    线稿 blur → 淡彩肖像 + 一句回忆（不是教训）

[3] 收束字卡（2s）
    「军训篇 · 暂告一段」/ 章节名
    试玩版可加：「大学还没过完。后面的遗憾，还在路上。」
```

### 3.3 揭面规则（章末 vs 全剧终）

| 时机 | 揭谁 | 不揭谁 |
|------|------|--------|
| **章末过场** | 本章内羁绊达标的 NPC | 未羁绊者保持虚线剪影；**主角永不揭面** |
| **全剧终** | 不再重复揭面；可汇总「你认识的人」缩略图 | 遗憾清单为主角 |

**原则**：章末揭面 = 「这段日子里，你记住了谁」；全剧终 = 「你错过了什么」。

### 3.4 军训篇 · 章末揭面候选

| 角色 | 本章羁绊条件 | 章末是否揭面 |
|------|-------------|-------------|
| 摇晃同学 | S1 选 A 或 B | ✓ |
| 排长 | S2 选 A 且互动 ≥2 | ✓ |
| 高个男生 | S2 点击 ≥3 | ✓ |
| 教官 | S3 自荐且通过，或 S3 互动教官 ≥3 | ✓ |
| 室友 | 跨子场景累计 ≥5（军训篇内） | ✓ |
| 主角 | — | ✗（全剧终后） |

未达标：过场中该位为空白线稿 +「你没看清 ta 的脸」。

### 3.5 过场动画（视觉）

- 风格：`visual_guide_v2.md` 实景 + 手绘  
- 手法：定镜 slow pan / 淡入淡出 / 单元素动效（哨子、空操场、远去的队列）  
- 时长：**克制**，不抢戏；目的是「换气」，不是 CG 短片

## 四、全剧终：遗憾清单（情感高潮）

### 4.1 定位

**整个游戏的最高潮是一张「遗憾清单」**——不是揭面，不是分数，不是成就。

- 让玩家看见：**你走的路、你没走的路、你可能错过的瞬间**  
- 目的：前置反思，减少真实大学里的同类遗憾（非恐吓）  
- 语气：陈述、留白；禁止「你应该」「你要学会」

### 4.2 试玩版（仅军训篇）vs 完整版

| 版本 | 高潮内容 |
|------|----------|
| **军训试玩** | **本章遗憾清单**（组合遗憾 + 选择遗憾 + 1 条通用）+ 预告更多章节 |
| **完整四年** | **跨章节累积遗憾清单** + 按时间线排列 + 隐藏路径遗憾 |

试玩版清单标题建议：**「军训 · 遗憾清单」**，避免与全剧终混淆。

### 4.3 清单结构（完整版）

```
┌─────────────────────────────┐
│        遗 憾 清 单           │
│   「如果重来，你会……吗？」    │  ← 可选副标题，可不要
├─────────────────────────────┤
│ [组合型] 跨章路径一句        │  ← 最贴玩家行为模式的一条
│ [章节型] 军训 / 百团 / …各1  │  ← 每玩过的大场景至少1条
│ [平行型] 「另一条路上……」    │  ← 未选分支里最有共鸣的一个瞬间
│ [通用型] 所有人都会触发的    │  ← 增加「原来大家都这样」
├─────────────────────────────┤
│ （滚动，每条 2～3 秒）        │
│ 无按钮打断，可点击跳过        │
└─────────────────────────────┘
        ↓
  「你一共认识了 N 个人」缩略（可选，非高潮）
        ↓
  主角揭面（仅全通后，可选）
        ↓
  重玩 / 分享清单卡片
```

### 4.4 清单文案原则

1. **具体**：「你后来知道他叫李明」>「你要多交朋友」  
2. **平行世界**：「如果你当时/report/没报告……」只呈现瞬间，不评判  
3. **可共鸣**：「你和 87% 的人一样选择了再睡一会儿」类数据慎用，要真实感不要营销感  
4. **与揭面分工**：清单写「错过」；揭面写「记住」——不在清单里重复揭面文案  

### 4.5 军训试玩 · 清单示例（保留现有逻辑）

- 组合型 1 条（如 A+A+通过 →「你一直站在最前面……」）  
- S1 选择专属 1 条  
- 通用 1 条（合影 / 不记得谁站旁边）  

详见 `scene_script_military_v4.md` 结算节；实现时从**章末过场**后进入**清单页**，而非在过场里混播清单。

## 五、单局体验时间线（军训试玩）

```
标题 → 开场文字
  → S1 探索/选择 → 余韵
  → 过渡文字
  → S2 …
  → 过渡文字
  → S3 …（含节奏关）
  → ★ 章末过场动画
  → ★ 本章羁绊揭面（依次）
  → ★ 遗憾清单（高潮，滚动）
  → 结尾字卡「军训篇 · 完」→ 重玩
```

**调整说明**（相对当前原型）：

| 现状 | 目标 |
|------|------|
| 结算页同时播遗憾 + 揭面 | **拆成**：过场揭面 → 清单高潮 |
| 揭面在清单后段 | 揭面提前到**过场**，清单单独成页 |
| 无过场动画 | 新增 5～15s interlude |

## 六、内容与「更真更全」的对应

每个大场景除剧情外，应让玩家**多知道一点真实的大学**（嵌入叙事，非科普弹窗）：

| 大场景 | 玩家可能新知道的 |
|--------|-----------------|
| 军训篇 | 集体节奏、身体极限、和「那个谁」的第一次连接 |
| 百团大战 | 社团 vs 学生会、面试话术、被选中/被无视 |
| 离家过节 | 第一次不回家、宿舍空一半、假装没事 |
| 抢课 | 教务系统、水课与硬课、信息差 |
| 期末突击 | 集体幻觉、通宵、「我能过」 |

写法：借 NPC 嘴、后果、遗憾清单呈现；**禁止**辅导员式总结。

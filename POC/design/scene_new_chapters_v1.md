## 章节2：百团大战

### 前置背景
- 十月的操场，上百个社团和学生会部门同时摆摊招新
- 你刚军训完，第一次觉得自己是"这个学校的人"
- 学长学姐热情地塞传单：吉他社、辩论队、学生会外联部、志愿者协会……
- 有人告诉你"学生会能加综测分"，有人说"社团才是真朋友"
- 你手里已经攥了十几张表，每张都要填"为什么想加入我们"
- 但你其实不知道自己想加入什么

### 情境
下午四点，招新最后一小时。你手里还剩两张表没填。吉他社的面试在六点，学生会外联部的面试在六点半，时间冲突了。一个学姐说"你只能选一个"。

### 核心选择节点

**1. 你填哪张表？**
- 吉他社 — 你真的想学吉他，但"没什么用"
- 学生会外联部 — "能锻炼能力"，但你不确定自己想锻炼什么
- 两张都交 — 学姐说只能选一个，你纠结到最后一刻
- 都不填 — 你转身走了，但走出十步又回头看了一眼

**2. 如果去了吉他社面试：**
- 你弹了一首歌，弹错了三个音 → 面试官说"有基础，来试试吧"
- 你不会弹 → "没关系，我们教零基础"
- 你弹得很好 → "你之前学过？"你犹豫要不要说实话

**3. 如果去了学生会面试：**
- 面试官问"你觉得外联部是做什么的" → 你答不上来
- 面试官问"如果商家拒绝赞助你怎么沟通" → 你编了一个答案
- 面试官看你简历 → 你发现你没什么可写的

**4. 面试后：**
- 被录取了 → 第一次例会你坐在角落，没人跟你说话
- 没被录取 → 你松了一口气，然后觉得有点失落
- 加入了但发现不喜欢 → 退出的尴尬比留下更难受

### 交叉情节：军训同学
- 面试时你看到一个军训时同排的同学，他也来了
- 他面试表现比你好——你想起他军训时当领队的事
- 如果你们都进了同一个部门，你们的关系从"军训同学"变成"同事"
- 跨标签影响：S1出头 → 面试时你更自信；S1隐形 → 你差点没交表

### 遗憾目录
- 你选了吉他社那条线上，你在社团晚会弹了一首歌，台下有人举手机拍你——你没走那条路
- 你选了学生会那条线上，你后来跟着部长去谈了一个商家的赞助，拿到了五百块——你没走那条路
- 你都没选的那条线上，你在回去的路上遇到了一个在操场弹吉他的人，你们聊了一晚上——你没走那条路

## 章节3：离家过节

### 前置背景
- 大一的第一个长假——中秋或国庆，只有三天
- 回家的车票涨价了，而且只够你在家里待一天
- 室友三个走了两个，剩你和另一个没回家的
- 宿舍楼变得特别安静，走廊里只有水管的声音
- 你妈早上打了电话来："在学校吃什么了？"你说"食堂有月饼"
- 其实食堂关门了，你打算吃泡面

### 情境
假期的第二天晚上，你在宿舍吃泡面。窗外有人在放烟花——不知道是留校的谁。你的手机响了一下，是家庭群里妈妈发的照片：一桌子菜，你的位置空着。

### 核心选择节点

**1. 这个晚上你怎么过？**
- 叫上室友去操场走走 → 你们看到有人在草地上弹吉他唱歌
- 一个人在宿舍看手机 → 你刷到高中同学的朋友圈，他们都在家
- 去教学楼自习 → 教室里只有一个人，是隔壁班的
- 给家里打个视频电话 → 你看到饭桌上的菜已经收了

**2. 如果去了操场：**
- 吉他的人在唱一首你高中时听的歌 → 你在旁边坐下来
- 有人在放烟花 → 你想起小时候和爸爸一起放过
- 室友说"要不我们去校门口吃烧烤" → 你们骑车去了

**3. 关于那个没回家的室友：**
- 他家里有情况（经济/关系），他不想说 → 你没问
- 他只是懒 → 你们一起打了一晚上游戏
- 他偷偷在阳台打电话，声音有点抖 → 你假装没听到

**4. 假期的最后一天：**
- 室友们陆续回来了，带了各种特产 → "我妈让我给你带的"
- 你什么都没带 → 你觉得有点不好意思
- 你妈寄了一个包裹，里面全是吃的 → 你分给室友

### 穿插小情节
- 辅导员在群里发"留校同学注意安全"，后面跟了个表情 → 没人回复
- 学校广播站放了一首歌，是整个假期唯一的声音
- 你去超市买东西，发现只剩你和一个阿姨 → 阿姨问你"怎么不回家"
- 楼下的快递柜满了——全是家里寄来的包裹

### 遗憾目录
- 你打了视频电话那条线上，你妈让你看桌上的月饼——"特意给你留的"。你没走那条路
- 你去了操场那条线上，弹吉他的人后来成了你大学里最好的朋友。你没走那条路
- 你室友打电话那条线上，他后来告诉你那是他最后一次和爷爷说话。你不知道

## 章节4：抢课

### 核心体验
教务系统崩溃的前一秒，所有人都在刷新页面。水课秒没，你想学的课只剩两个名额。你女朋友说"选那门水课吧，我们一起"，但你想选的那门课是你真正感兴趣的。

### 关键选择
- 抢水课还是抢想学的课
- 帮不帮室友代抢
- 选上了/没选上后的连锁反应
- "退课/换课"的纠结期

### 独有性
- 只有大学才有教务系统选课的体验
- 信息不对称（学长知道哪个老师给分好）
- 集体焦虑（所有人都在抢，你不知道自己在抢什么）

## 章节5：期末突击

### 核心体验
一学期没听的课，考前一周开始学。图书馆从早到晚都是人。你制定了完美的复习计划，但第一天就刷了三个小时手机。

### 关键选择
- 突击哪门课先
- 要不要找人借笔记
- 通宵还是早起
- 考完发现考的和复习的不一样

### 独有性
- "考前一周学一学期"是大学集体幻觉
- 图书馆占座文化
- 考试时的集体崩溃和集体放松

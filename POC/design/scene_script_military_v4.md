### 选择定义

#### 选择A：报告教官！
- **标签**：bold +2
- **触发**：点击同学脸部第2次 → 浮现选项 → 确认
- **画面变化**：教官转身看向你，全排的人头转向你
- **叙事**：
```
教官转身看向你。
全排的人头转向你。
你喊了声"报告"。声音比你预想的大。
```
- **余韵**：
  - 2秒后 → 室友小声说"你真喊了啊"
  - 4秒后 → 过渡文字「二十分钟后，解散哨响了」
  - 6秒 → 切场景2

#### 选择B：悄悄撑他一把
- **标签**：support +2
- **触发**：点击同学身体第2次 → 浮现选项 → 确认
- **画面变化**：你的姿势歪了，隔壁教官瞄了一眼
- **叙事**：
```
你的手搭上了他的胳膊。
你的姿势歪了。隔壁教官瞄了一眼。
他没看你。但他站稳了。
```
- **余韵**：
  - 2秒后 → 同学看了你一眼，嘴唇动了动，没说出话
  - 4秒后 → 过渡文字「二十分钟后，解散哨响了」
  - 6秒 → 切场景2

#### 选择C：往旁边挪半步
- **标签**：quiet +2
- **触发**：点击地面第3次 → 浮现选项 → 确认
- **画面变化**：你的脚位移了半步，你和同学之间的距离变了
- **叙事**：
```
你的脚位移了半步。
你和同学之间的距离变了。
你没碰到他。你也没走远。
```
- **余韵**：
  - 2秒后 → 你的影子和他的影子重叠了一点
  - 4秒后 → 过渡文字「二十分钟后，解散哨响了」
  - 6秒 → 切场景2

#### 超时（30秒无任何有效操作）
- **标签**：quiet +2（等同选择C）
- **叙事**：
```
同学自己站稳了。
但后来他又晃了一次，倒了。

画面变暗。
```

### ⭐ 节奏游戏：走正步

**触发**：玩家选择自荐后进入

#### 开始画面

```
你举了手。
教官看了你一眼。
"行。走两步。"
```

#### 游戏过程

```
┌──────────────────────────────────┐
│                                  │
│   🚶 (你的角色侧面，走步动画)     │
│                                  │
│  ─ ─ ─ ● ─ ─ ─ ─ ● ─ ─ ─      │  ← 节拍线从右向左滚动
│              ↓                   │
│          【击！】                 │  ← 打击判定区
│                                  │
│   得分: ●●●○○○○  连击: 3        │
│   进度: ████████░░░░░░░░  8/16  │
└──────────────────────────────────┘
```

**参数**：
- 16个节拍，间隔850ms
- Perfect ±80ms，Good ±160ms，超出=Miss
- Perfect+Good ≥ 12 通过

**判定反馈**：
- **Perfect**：角色步伐有力 + 打击区闪光 + 粒子效果
- **Good**：还行，稍微歪了一点
- **Miss**：角色踉跄/顺拐 + 旁边同学偷笑 + 画面轻微抖动
- **连续3个Perfect**：教官微微点头
- **连续3个Miss**：对面排有人指指点点
- **最后3个全Perfect**：画面变亮，角色走得特别自信
- **最后3个全Miss**：角色开始顺拐，画面轻微变暗

#### 通过（Perfect+Good ≥ 12）

> 画面闪白光 → "PASS"

```
教官："就你了。走在最前面。"
```

> 2秒 → 你的角色从方阵中走到最前面

> 2秒 → 室友在下面小声说了什么。你听不清。

> 2秒 → 教官："明天早上六点半，操场加练。"

> 你的角色回到队列。

> 3秒 → 过渡文字：

```
那天晚上你练了很久的正步。在宿舍走廊。
室友说"你别练了，脚步声太大"。
你没停。
```

> 5秒 → 切结算页

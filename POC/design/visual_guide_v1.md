# 视觉设计指南 v1

> ⚠️ **已 superseded**：请以 [`visual_guide_v2.md`](./visual_guide_v2.md) 为准（定调：实景 + 手绘，无五官，羁绊揭面）。  
> 下文 3D Q 版盲盒内容仅作历史参考。

## 核心风格

**3D Q版渲染 + 中国大学军训实景**

### 人物风格
- 3D Q版渲染风格（参考泡泡玛特盲盒质感，参考图见 `concept_art/IMG_1457.PNG`）
- 完全光滑圆润头部，**无任何五官特征**（无眼、无嘴、无鼻）
- 淡黄色中国人肤色（#FDE8D0 近似值）
- 小小的身体，比例偏Q版
- **不带披风/斗篷**，**不带发光边缘**，**不带光晕特效**
- 身体通过姿态、动作传达情绪（不是表情）
- 有头发轮廓（从帽子边缘/帽檐下露出少量发丝暗示性别）

### 服装规范（必须严格）

#### 参训学生
- **上衣**：07式/19式绿色丛林迷彩作训服（上衣），翻领，前胸两个贴袋
- **裤子**：同款迷彩作训裤，直筒
- **帽子**：绿色作训帽（平顶软帽，帽檐平直），正面有小型五星红旗帽徽（红色底+五颗黄色星）
- **鞋**：黑色作训鞋/迷彩胶鞋
- **配饰**：无腰带（夏季训练），部分人可能有白色T恤领口露出

#### 教官
- 同款迷彩作训服，但更整洁
- 作训帽（同款，小型五星红旗帽徽一致）
- 可能佩戴哨子（挂在胸前）
- 站姿笔直，和学生的松散形成对比

#### 关键区分
- 教官 vs 学生：不是靠衣服颜色不同，而是靠**站姿、气质、配件（哨子）**
- 不同排的学生：服装完全一样，区分靠**位置、名字贴、小配件**（如排长有红色臂章）

## 场景美术要求
- 背景偏写实/轻插画混合（不用光遇那种完全抽象的渐变天空）
- 操场、草地、跑道等环境要真实可信
- 光线偏暖（夏日感），有光影层次
- 光粒子可以有，但要克制——不要满屏都是
- 整体色调：暖黄/暖橙为主，搭配军训的绿色

## ⭐ 核心设计：羁绊揭面机制

### 原则
- **故事进行中**：所有人物无明显五官，光遇风格
- **结算页/故事结束后**：与玩家建立了"羁绊"的角色，面容通过动画浮现
- **主角（玩家）**：面容在整部游戏完全通关后才揭示

### 羁绊判定规则

每个场景的关键NPC都有一个隐藏的"羁绊值"，通过与该角色的互动次数和选择累积：

| 角色 | 羁绊触发条件 | 揭面时机 |
|------|------------|---------|
| 摇晃同学 | S1选了A(报告教官)或B(撑一把) | S1结算时 |
| 排长 | S2选了A(领唱)且互动排长≥2次 | S2结算时 |
| 高个男生 | S2点击高个男生≥3次 | S2结算时 |
| 教官 | S3自荐+节奏游戏通过，或S3全程互动教官≥3次 | S3结算时 |
| 室友 | 跨场景累计点击室友≥5次 | 最终结算时 |
| 主角(你) | 完成全部章节 | 全游戏通关后 |

### 揭面动画设计

**单个角色揭面**：
1. 结算页展示该角色的光遇风格剪影
2. 面容从模糊到清晰，缓慢浮现（1-2秒动画）
3. 配一句与该角色相关的遗憾/回忆文案
4. 如果未建立羁绊：该角色位置是空白剪影+虚线轮廓，配文"你没看清ta的脸"

**主角揭面**：
1. 所有章节结算完成后
2. 一面镜子/水面/手机屏幕的意象
3. 面容极缓慢浮现（3秒），最后定格
4. 配文："你终于看清了自己"
5. 这是整个游戏的情感高潮

### 揭面的情感设计

- 揭面不是"奖励"，是"记忆"
- 你不帮那个同学，就不会看到他的脸——不是惩罚，是因为你和他之间没有故事
- 每次通关，揭面的角色组合不同→重玩意愿
- 主角的脸是最后的彩蛋→完整通关的动力

### 与产品名的呼应

"那个谁"——
- 军训里你记不住每个人的名字
- 你只记得"那个快晕的同学"、"那个领唱跑调的排长"、"那个不敢领唱的高个男生"
- 直到你和某个人产生了连接，ta才从"那个谁"变成了一个具体的人
- **这就是产品的核心表达：人与人的连接，让模糊变清晰**

## UI 框架参考

### 整体布局（竖屏 9:16）
```
┌──────────────────────┐
│ 场景标签    引导提示   │ ← 顶部信息栏（小字，不抢戏）
│                      │
│                      │
│    【场景插画区】      │ ← 占60-70%画面
│    背景+角色          │
│                      │
│                      │
├──────────────────────┤
│  互动反馈/叙事文字     │ ← 底部叙事区（可滚动）
│  [选项按钮]           │
└──────────────────────┘
```

### 交互元素样式
- 可点击的角色：微弱的呼吸动画（scale 1.0↔1.03，2秒循环）
- 选项按钮：暖橙边框，圆角，按下变实心
- 叙事文字：米白色，14px，逐字淡入
- 过渡动画：0.5秒交叉淡入淡出

### 配色方案
- 背景：根据场景变化（操场=暖黄，傍晚=暖橙，夜晚=深蓝）
- 文字：#e8e0d4（米白）
- 强调/交互：#c4956a（暖橙）
- 选项按钮：边框#c4956a，文字#c4956a，hover变实心
- 不可交互元素：低饱和度，降低opacity

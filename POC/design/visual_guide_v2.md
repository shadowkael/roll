## 一、一句话风格

**九月操场上的老照片，上面叠了你在日记边距里画的小人——没有脸，直到你终于想起那个人。**

---

## 二、必须守住的核心（不可妥协）

以下与具体渲染技法无关，是 Roll /「那个谁」的视觉宪法：

| 原则 | 说明 |
|------|------|
| **过程中无五官** | 故事进行中，所有角色**没有可辨认的五官** |
| **姿态传达情绪** | 靠站姿、头倾、肩线、手臂、距离表达状态，不靠表情 |
| **羁绊才揭面** | 只有与玩家建立连接的角色，结算时才浮现清晰面容 |
| **主角最后看清** | 玩家自己的脸，在完整体验结束后才揭示 |
| **真实军训情境** | 场景、服装、光线可信；克制、不煽情、不游戏化 |

> 揭面不是「奖励」，是「记忆」——你没和那个人产生连接，就不该看清 ta 的脸。

## 三、正式风格：实景 + 手绘

### 3.1 定义

**实景层**：大学军训环境的写实底图（照片级或轻度插画化的写实场景）。  
**手绘层**：叠在实景上的线稿/淡彩人物与关键物件，**无五官**。  
**交互层**：透明热区、高亮、叙事 UI（不抢戏）。

```
┌─────────────────────────────┐
│  [L4] UI：叙事 / 选项         │
├─────────────────────────────┤
│  [L3] 交互热区 + 高亮反馈     │
│  [L2] 手绘人物（线稿/淡彩）   │  ← 无五官，可切换姿态
│  [L1] 实景底图（操场/阳光）   │  ← 暖色、略褪色、有生活感
└─────────────────────────────┘
```

### 3.2 为什么选这个方向

- 比 3D Q 版盲盒更贴「真实、怀旧、像真的经历过军训」
- 比光遇式抽象更有**大学**和**年代感**
- **无五官**在线稿/背影/剪影上更自然，不像「模型没做完」
- 揭面时从线稿 → 带五官的肖像，**对比强、情感准**
- 无专业美术时：实景可 AI/图库，人物可 AI 线稿或统一描边，比统一 3D 风更易量产

### 3.3 已废弃/仅作内部使用的形式

| 形式 | 状态 |
|------|------|
| CSS / 简笔画火柴人 | ❌ 仅工程原型，不对玩家展示 |
| 3D Q 版 / 泡泡玛特盲盒 | ⚠️ 备选，非目标风格 |
| 光遇式全抽象、无环境 | ❌ 不符合「军训实景」 |
| 全程精致立绘 + 大眼睛 | ❌ 变成恋爱 AVG，破坏揭面机制 |

## 四、画面分层规范

### 4.1 实景底图（L1）

**内容**：操场、塑胶跑道、教学楼远景、九月阳光、队列的空镜或远景人影（可模糊）。  
**气质**：像手机相册里调过色的军训照片——**暖黄、轻颗粒、略褪色**，不要赛博、不要日系高光。  
**禁止**：奇幻天空、满屏光粒子、与军训无关的元素。

**光线参考**：

| 场景 | 光线 |
|------|------|
| S1 站军姿 | 清晨～上午，阳光从一侧打来，影子长 |
| S2 拉歌 | 临近中午，偏暖黄，略晃眼 |
| S3 领队选拔 | 午后，影子更长，气氛略紧 |

### 4.2 手绘人物层（L2）

**技法**：线稿为主，可极淡水彩/单色填充；线条允许**略抖、不完全均匀**（日记感）。  
**头部**：光滑轮廓或帽檐遮住上半脸，**绝不画眼口鼻**。  
**身体**：正常比例或略缩短（5～6 头身），**不要 Q 版盲盒比例**。  
**人数**：每镜头 3～5 个清晰角色即可，其余远景可省略或化为色块。

**情绪靠姿态（示例）**：

| 状态 | 姿态语言 |
|------|----------|
| 快晕 | 头歪、肩沉、重心偏移 |
| 紧张 | 肩耸、手握拳、脚并拢 |
| 想出头又怂 | 手半抬、身体前倾又收 |
| 教官 | 背直、脚开、胸哨；与学生松散对比 |
| 你（主角） | **默认背影或侧面**，不正面直视镜头 |

**服装**（手绘层须可辨认，但不必精细渲染）：

- 07/19 式丛林迷彩作训服、绿色平顶作训帽、小型红旗帽徽  
- 教官：同款但更笔挺 + 胸前哨子  
- 排长：红臂章；同学：帽檐名字贴（可只写「李」等一字）

**线稿色值建议**：`#7a756c`（暖灰褐）或 `#5a6b52`（淡军绿线），透明底 PNG。

### 4.3 交互层（L3）

- 可点击区域：**透明热区**，必要时显示暖橙描边（`#c4956a`）呼吸动画  
- 点击反馈：轻微 scale、局部 dim 其他区域、短气泡  
- 选择按钮：从**被点击对象附近**弹出，而非固定底部三按钮（与交互设计 v3 一致）  
- **定镜**：场景切换时可全屏 crossfade 实景图（现有 Establishing Shot 保留）

### 4.4 UI 层（L4）

- 背景 UI 区：`#1a1a1a` 或半透明黑底，不遮挡画面主体  
- 文字：`#e8e0d4`，短句、口语化  
- 强调/可交互：`#c4956a`

## 五、章末过场 · 羁绊揭面（视觉）

逻辑见 `product_experience_v1.md`。视觉上在**每个大场景结束**的过场动画后播放，**不是**在遗憾清单里混播。

### 5.1 故事中

- 手绘人物始终无五官  
- 可强调：帽檐、名字贴、轮廓——「好像有个『李』字，但看不清」

### 5.2 结算页 · 单个角色

**已羁绊**：

1. 展示该角色手绘剪影（或实景中小窗裁切 + 线稿）  
2. 面部区域：`blur(12px) → blur(0)` 或线稿上**缓慢浮现**五官（1.5～2s）  
3. 配一句与该角色相关的遗憾/回忆  
4. 五官风格：**温和、写实的淡彩肖像**，与线稿同一人物，不要突然变 3D 盲盒

**未羁绊**：

- 空白线稿轮廓 + 虚线脸廓  
- 文案：「你没看清 ta 的脸」

### 5.3 主角揭面

- 意象：镜子 / 手机屏反光 / 操场积水倒影  
- 面容在反射面中极慢浮现（约 3s）  
- 文案：「你终于看清了自己」

### 5.4 羁绊判定（不变）

| 角色 | 羁绊条件 | 揭面时机 |
|------|----------|----------|
| 摇晃同学 | S1 选 A 或 B | 军训篇·章末过场 |
| 排长 | S2 选 A 且互动排长 ≥2 | 军训篇·章末过场 |
| 高个男生 | S2 点击高个 ≥3 | 军训篇·章末过场 |
| 教官 | S3 自荐且 rhythm 通过，或 S3 互动教官 ≥3 | 军训篇·章末过场 |
| 室友 | 军训篇内累计点击室友 ≥5 | 军训篇·章末过场 |
| 主角 | 全剧所有章节完成 | 遗憾清单之后（可选） |

> 全剧情感高潮是**遗憾清单**，不是揭面。详见 `product_experience_v1.md`。

## 六、布局与平台

### 6.1 画幅

- **正式画幅**：横屏 **16:9（1920×1080）**，与强制横屏 H5 一致  
- **BG / 定镜**：按横屏生产；**SK 角色**为透明立绘（单人竖构图叠放）  

### 6.2 布局示意

```
┌────────────────────────────────────────────┐
│ 场景标签                         引导提示   │  顶栏
│                                            │
│            【实景 + 手绘区 · 横屏】          │  主视觉
│                                            │
├────────────────────────────────────────────┤
│           叙事 / 气泡 / 选项                 │  底部叙事区
└────────────────────────────────────────────┘
```

## 七、与产品名的关系

「那个谁」——

- 军训里记不住每个人的名字和脸  
- 你只记得「那个快晕的」「那个想领唱又不敢的」  
- **无五官不是偷懒，是「还没记住」**  
- 揭面 = 某次连接被想起来 = 从「那个谁」变成具体的人  

---

## 八、制作路线（给无美术团队）

1. **定风格板**：同一场景三联——① 纯实景 ② 实景+线稿无脸 ③ 揭面后肖像  
2. **P0 资产**：3 张实景底图 + 4 个核心角色线稿（各 2 姿态）  
3. **引擎**：定镜全屏 + 热区 + 线稿 sprite 叠加（见 `visual_asset_brief_v3.md`）  
4. **淘汰**：玩家可见的 CSS 简笔画层  

---

## 九、参考 mood（非临摹）

- 纪实摄影 + 手账插图  
- 淡色老照片 + 铅笔速写  
- **不要**：泡泡玛特、光遇、Galgame 立绘、简笔画占位
